TensorFlow 2.19 Setup Guide (Python 3.10 + venv)
1.
	Install Python 3.10
	Download and install Python 3.10 from https://www.python.org/downloads/release/python-310x/
	During install, check "Add Python to PATH".
2.
	Create a Virtual Environment
	Open a terminal or command prompt and run:

		C:\Users\YourName\AppData\Local\Programs\Python\Python310\python.exe -m venv tf310

	Then activate it:

		.\tf310\Scripts\activate

	You should now see (tf310) at the beginning of your prompt.

  	Check Python and Fix pip (if needed)

		python --version
		pip --version

	If pip gives an error or is missing, run:

		python -m ensurepip --upgrade
		python -m pip install --upgrade pip
3.
	Install TensorFlow 2.19.0

		pip install tensorflow==2.19.0

4.
	To check tensorflow:
		python -c "import tensorflow as tf; print(tf.constant(2.0) * tf.constant(-140.5))"
	To check GPU installation:
		python -c "import tensorflow as tf; print(tf.config.list_physical_devices('GPU'))"
